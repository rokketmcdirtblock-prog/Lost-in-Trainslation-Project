# BP_HTML_08_Nodejs

A hands-on repository to explore the basics of HTML, CSS, Tailwind and Node.js.

## About this repository

This repo illustrates how to build a simple web project using HTML and CSS for the front-end, and Node.js for the backend. It’s intended as a learning/demo project for students and developers getting started.


## Topics covered

- HTML — structure and semantics  
- CSS — styling and layout  
- Node.js — a basic server setup (`server.js`)  
- Static assets (under `public/`)  
- Project structure: `package.json`, `.gitignore`, `server.js`, `public/` folder  

## Getting started

### Prerequisites

- Node.js (v14.x or newer recommended)  
- npm (comes bundled with Node.js)
